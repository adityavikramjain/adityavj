## Option 1: "Ink & Ivory" - Editorial Minimalism
**Vibe**: New Yorker magazine meets brutalist design
**Who does this**: Almost nobody in tech (that's the point)

```css
:root {
    /* Primary accent - Deep Ink Blue (replaces orange) */
    --primary: #1a1f3a;
    --primary-light: rgba(26, 31, 58, 0.08);
    --primary-glow: rgba(26, 31, 58, 0.2);

    /* Background - Warm Ivory */
    --background: #f9f7f4;

    /* Text */
    --charcoal: #1a1f3a;

    /* Accent - Burgundy for rare pops */
    --accent: #8b2635;
}
```

**Why it works**:
- Timeless, not trendy
- Feels like a design studio, not a startup
- High contrast = accessibility win
- Unexpected for AI/tech space

**Aesthetic feel**: Kinfolk magazine, Apartamento, The Economist
